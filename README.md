# Relatório de Desempenho de Campanhas do Facebook Ads via WhatsApp

Este projeto permite que você colete dados de campanhas do Facebook Ads e envie um relatório de desempenho via WhatsApp utilizando a API do Twilio.

## Requisitos

Antes de começar, você precisará instalar os seguintes pacotes:

- **Python 3.x**: Certifique-se de que o Python 3.x está instalado em sua máquina.
- **pip**: O gerenciador de pacotes do Python, que normalmente vem junto com o Python.

- *Precisará de uma conta na Twilio / Criar um App No Meta Developers*


### Bibliotecas Python

As seguintes bibliotecas Python são necessárias:

- `facebook_business`: Biblioteca oficial do Facebook para acessar a API de anúncios.
- `pandas`: Biblioteca para manipulação de dados.
- `twilio`: Biblioteca para enviar mensagens via API do Twilio.
- `python-dotenv`: Biblioteca para resguardar credenciais importantes.


Para instalar as bibliotecas necessárias, você pode usar o seguinte comando:

```bash
pip install facebook-business pandas twilio python-dotenv
